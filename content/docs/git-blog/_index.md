## Introduction

백엔드 개발을 중심으로 데이터 처리와 시스템 아키텍처 설계에 관심이 있습니다.  
Python 기반의 FastAPI 환경에서 API 서버를 설계·개발하며, 비동기 처리와 안정적인 서비스 운영을 중요하게 생각합니다.

Elasticsearch 기반 검색 시스템, 데이터 파이프라인 구축, 성능 최적화 경험이 있으며  
Docker 및 컨테이너 환경에서의 운영과 확장에도 익숙합니다.

단순한 기능 구현을 넘어,  
**유지보수성**, **확장성**, **명확한 구조**를 갖춘 시스템을 설계하고 구현하는 것을 목표로 합니다.
